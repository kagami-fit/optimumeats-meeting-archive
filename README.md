# OptimumEats Minutes Archive

OptimumEatsの店舗運営マニュアル化プロジェクト用に作成した、閲覧専用の議事録アーカイブサイトです。

## 使い方

```bash
npm install
npm run dev
```

起動後、以下を開きます。

```text
http://localhost:3000
```

## 内容

- 日付別の議事録一覧
- 要約、議題、決定事項、課題、次回アクション
- 担当者・期限・状態つきのアクション項目
- OptimumEats公式ロゴとトンマナに合わせた画面
- Image-gen2で作成した各MTGの図解
- 2026年7月5日・7月6日の議事録4件を掲載

詳細は `ABOUT.md` を参照してください。
